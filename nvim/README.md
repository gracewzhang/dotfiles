# Neovim Configuration

Primarily set up for Python and Typescript development. Most of the plugin configs are taken from [Launch.nvim](https://github.com/LunarVim/Launch.nvim), [jdhao's neovim-config](https://github.com/jdhao/nvim-config), and [ntk148v's neovim-config](https://github.com/ntk148v/neovim-config).
